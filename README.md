## 🗳️ VoteBot India – Election Process Education AI

## 🚀 Overview
VoteBot India is an AI-powered assistant designed to educate citizens about the election process in India. It simplifies complex procedures into easy-to-understand responses for first-time voters.

---

## 🎯 Problem
Many citizens, especially first-time voters, lack clarity about:
- How to vote
- Voter ID process
- Eligibility criteria

---

## 💡 Solution
An interactive AI chatbot that:
- Answers election-related queries
- Provides step-by-step guidance
- Supports English and Hindi responses
- Improves civic awareness

---

## ⚙️ Features
- 🗳️ Voting process explanation
- 🆔 Voter ID guidance
- ✅ Eligibility checking
- 🤖 AI-based question answering
- 🌐 Multilingual support (English + Hindi)
- ⚡ Fast API-based backend

---

## 🧠 Tech Stack
- Python (FastAPI)
- HTML/CSS (Frontend UI)
- OpenRouter AI (LLM Integration)
- Prompt Engineering
- Docker
- Google Cloud Run (Deployment)

---

## ☁️ Deployment
This project is containerized using Docker and deployed on **Google Cloud Run** for scalability and performance.

---

## 🔐 Security
- Sensitive API keys are stored using environment variables (.env)
- `.env` file is excluded using `.gitignore`

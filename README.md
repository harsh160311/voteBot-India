# 🗳️ VoteBot India – Election Process Education AI

## 🚀 Overview
VoteBot India is an AI-powered assistant designed to educate citizens about the election process in India. It simplifies complex procedures into easy-to-understand responses for first-time voters.

## 🎯 Problem
Many citizens, especially first-time voters, lack clarity about:
- How to vote
- Voter ID process
- Eligibility criteria

## 💡 Solution
An interactive AI chatbot that:
- Answers election-related queries
- Provides step-by-step guidance
- Improves civic awareness

## ⚙️ Features
- Voting process explanation
- Voter ID guidance
- Eligibility checking
- AI-based question answering

## 🧠 Tech Stack
- Python (Flask)
- HTML/CSS
- Prompt-based AI logic
- Google Cloud Run (Deployment)

## ☁️ Deployment
Deployed using Google Cloud Run for scalable access.

## 🔐 Security
Sensitive data handled via environment variables (.env)

## 📌 Future Scope
- Multilingual support (Hindi + regional languages)
- Integration with real government APIs
- Voice assistant support
